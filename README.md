# FilmScan Pro

[中文]
FilmScan Pro 是一款专为底片翻拍设计的移动端 Web 工具。它可以将手机拍摄的负片实时转为正像，并提供色彩校正功能，方便摄影爱好者快速查看胶片拍摄效果。

**主要功能：**
*   **实时显影**：将负片反转为正常色彩，实时预览。
*   **自动锁定**：进入负片模式时自动锁定曝光和对焦，防止画面跳变。
*   **色罩平衡**：支持一键自动采样或手动调节，去除底片橙色色罩。
*   **即开即用**：纯 Web 技术，无需安装 App，支持全屏沉浸操作。
*   **隐私安全**：所有图像处理均在**本地设备**完成，没有云端上传，保护您的创作隐私。

---

[English]
FilmScan Pro is a mobile web tool designed for film scanning and visualization. It converts film negatives into positive images in real-time and provides essential color correction features.

**Key Features:**
*   **Real-time Visualization**: Instantly inverts negatives to positive colors.
*   **Auto-Lock**: Automatically locks exposure, white balance, and focus in negative mode to ensure stability.
*   **Mask Balancing**: Supports automatic sampling or manual adjustment to remove the orange film mask.
*   **Ready to Use**: Browser-based, no installation required, supporting full-screen immersive mode.
*   **Security & Privacy**: All image processing is performed **locally on your device**. No data is uploaded to any server.
